Defined in:
manifests/init.pp

Summary

Fully manages postfix.

Overview

Class: postfix

This module fully manages postfix. It can install and uninstall postifx, configure or delete every vendor and custom configuration file, and optionally manage the postfix service.

Examples:

Minimum configuration, sufficient for vendor-specified defaults
